# STM32 智能环境监控系统

这是一个基于 STM32F103C8T6 的智能环境监控系统。系统通过 DHT11 (温湿度)、BH1750 (光照)、土壤湿度传感器实时监测环境，并支持通过串口指令配置水泵和蜂鸣器的自动控制规则。

## 主要特性
- **实时监测**: 集成多种传感器数据采集。
- **动态阈值**: 支持通过串口指令实时修改控制阈值。
- **断电保存**: 配置的规则自动保存至内部 Flash，重启不丢失。
- **强制控制**: 支持无视传感器数据的强制开关模式。
- **抗干扰设计**: 针对水泵启动瞬间的电磁干扰进行了软件保护。

## 使用手册

详细的使用说明请参考：[入门使用手册](快速入门_readme.txt)

### 快速开始 (串口指令摘要)
- **查看规则**: `ls cmd`
- **设置规则**: `PUMP Soil 0 low 30 high 1` (当土壤湿度 0-30% 时开启水泵)
- **强制开启**: `PUMP 1`
- **复位自动**: `PUMP auto`
- **帮助信息**: `help`
